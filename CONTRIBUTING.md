# Contributing

Thanks for your interest in **evm-tools**.

This project is developed and maintained internally by Daxchain. It is **closed
to outside contributions**. The repository is public so the tools can be
installed via Homebrew and the `curl` installer and so others can read and learn
from the code — not as an invitation to contribute.

What that means in practice:

- **Pull requests** from outside the Daxchain organization will not be reviewed
  or merged. Please don't open them.
- **Feature requests and bug reports** from the public are not tracked here and
  may be closed without response.
- **Forks** are welcome under the [Apache-2.0 license](LICENSE). You're free to
  fork and adapt the code for your own use; just don't expect changes to flow
  back upstream.

## Reporting a security issue

Security reports are the one channel we do want from outside. **Do not** open a
public issue or pull request for a vulnerability — see [SECURITY.md](SECURITY.md)
for how to report one privately.
